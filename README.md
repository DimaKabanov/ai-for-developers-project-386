## Live Demo

**Deployed on Render.com:** [https://calendar-booking-pzgn.onrender.com](https://calendar-booking-pzgn.onrender.com)

Calendar Booking - приложение для бронирования встреч с возможностью выбора временных слотов.

### Features
- Просмотр доступных типов событий
- Бронирование встреч на свободные слоты
- Админ-панель для управления событиями

### Tech Stack
- **Frontend:** React 19 + TypeScript + Vite + Mantine UI
- **Backend:** Ruby on Rails 7 API-only
- **Deployment:** Docker on Render.com
